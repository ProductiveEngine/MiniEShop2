# MiniEShop 2
This is a simple web application showcasing the use of Angular JS 2.

#IDE
JetBrains WebstStorm

#Technologies
* Google AngularJS v2
* Bootstrap v3.3.7

#Getting started
1. Download npm
2. Open cmd and go to the project directory
3. Run npm install
4. Run npm start
